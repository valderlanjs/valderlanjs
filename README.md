## Projeto Principal — Plataforma de BI

Desenvolvimento de uma plataforma de BI proprietária do zero, cobrindo todas as camadas de uma arquitetura de dados real: da ingestão até a visualização.

```
ERP Simulado (Northwind)
        │
        ▼
    [ Stage ]  ──►  [ Data Warehouse ]  ──►  [ Mart ]
        │                                        │
        └──────── Apache Airflow (DAGs) ─────────┘
                                                 │
                                            [ FastAPI ]
                                                 │
                                            [ React ]
```

| Camada | Tecnologias |
|---|---|
| Orquestração | Apache Airflow 2.9.1 · Docker Compose |
| Banco de Dados | PostgreSQL 16 · Neon Cloud (colaborativo) |
| Backend / API | FastAPI · SQLAlchemy · Alembic · Redis |
| Segurança | JWT · RBAC · Rate Limiting (SlowAPI) · HttpOnly Cookies |
| Frontend | React 19 · Vite · Tailwind CSS v3 · React Router 7 |
| Modelagem | Star Schema — `dim_*` + `fact_*` |

📁 [`project_bi_engining`](https://github.com/valderlanjs/project_bi_engining) · [`project_data_engineering`](https://github.com/valderlanjs/project_data_engineering)
