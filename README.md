## 🚀 Projeto Principal — Plataforma de BI

> Desenvolvimento de uma plataforma de BI própria do zero, cobrindo todas as camadas de uma arquitetura de dados real em produção.

```
ERP (Northwind) ──► Stage ──► Data Warehouse ──► Mart ──► FastAPI ──► React
                        └── Apache Airflow (orquestração) ──────────┘
```

| Camada | Tecnologias |
|--------|-------------|
| **Orquestração** | Apache Airflow 2.9.1 + Docker Compose |
| **Banco de Dados** | PostgreSQL 16 + Neon Cloud (colaborativo) |
| **Backend / API** | FastAPI + SQLAlchemy + Alembic + JWT + Redis |
| **Frontend** | React 19 + Vite + Tailwind CSS + React Router 7 |
| **Modelagem** | Dimensional (Star Schema) — `dim_*` + `fact_*` |
| **Segurança** | RBAC + Rate Limiting (SlowAPI) + HttpOnly Cookies |

🔗 Repositórios: [`project_bi_engining`](https://github.com/valderlanjs/project_bi_engining) · [`project_data_engineering`](https://github.com/valderlanjs/project_data_engineering)
